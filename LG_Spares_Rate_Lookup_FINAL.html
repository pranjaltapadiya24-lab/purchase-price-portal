One-time setup needed
  1. 1 Open Google Drive and upload each of your 3 Excel files — Drive will convert them to Google Sheets automatically.
  2. 2 Open each Google Sheet → click Share → set access to "Anyone with the link can view". Then copy the long ID from the URL (between /d/ and /edit).
  3. 3 Open this HTML file in Notepad or VS Code. Find the SHEET_CONFIG section near the top of the <script> block. Paste each Spreadsheet ID into the matching slot.
  4. 4 Save the file and open it in Chrome. Prices will now load live — every time your father edits a cell, the portal reflects it on the next refresh.
If you see CORS or fetch errors, host this file on GitHub Pages (same as your order portal) — that fixes it permanently.
